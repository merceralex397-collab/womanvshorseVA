# Implementation Artifact — CORE-006: Create enemy variants (brown, black, war horse, boss)

## 1. Summary

Created four enemy variant scripts extending `EnemyBase`, each with unique visual and stat profiles. Updated `wave_spawner.gd` to use a factory pattern with `match` statement to instantiate the correct variant by type name.

### `scripts/enemy_black.gd`
Black enemy variant — fast (150 speed), 1HP, with speed lines visual effect when moving.
```gd
class_name EnemyBlack
extends EnemyBase

var _last_position: Vector2 = Vector2.ZERO

func _ready() -> void:
	super()
	type = "black"
	body_color = Color(0.2, 0.2, 0.2)
	body_size = Vector2(25, 35)
	speed = 150.0
	max_health = 1
	health = max_health

func _physics_process(delta: float) -> void:
	_last_position = global_position
	super(delta)

func _draw() -> void:
	super()
	if velocity.length() > 50:
		var behind_offset: Vector2 = -Vector2.from_angle(rotation) * body_size.x * 0.6
		for i in range(3):
			var line_start := _last_position + behind_offset + Vector2.from_angle(rotation + PI/2) * (i - 1) * 8
			var line_end := line_start + (-velocity.normalized() * body_size.x * 0.4)
			draw_line(line_start - global_position, line_end - global_position, Color(0.4, 0.4, 0.4), 2.0)
```
Speed lines are drawn behind the body when velocity > 50, using 3 parallel lines offset perpendicular to movement direction.

### `scripts/enemy_boss.gd`
Boss variant — largest body (50x65), speed 100, 10HP, with pulsing gold modulate effect.
```gd
class_name EnemyBoss
extends EnemyBase

func _ready() -> void:
	super()
	type = "boss"
	body_color = Color(1.0, 0.84, 0.0)
	body_size = Vector2(50, 65)
	speed = 100.0
	max_health = 10
	health = max_health

func _process(delta: float) -> void:
	var t: float = sin(Time.get_ticks_msec() / 300.0) * 0.5 + 0.5
	modulate = Color(1.0, 0.84 + t * 0.16, 0.0 + t * 1.0, 1.0)
```
Pulsing period is ~2 seconds using sine wave. The modulate is set each frame but does not interfere with damage flash since `take_damage()` in `EnemyBase` sets `modulate` to a white flash color during the 0.15s flash window.

## 3. Files Modified

### `scripts/wave_spawner.gd`
Updated `_create_enemy(type: String) -> EnemyBase` to use factory pattern:

**Before:**
```gd
func _create_enemy(type: String) -> Node2D:
	var enemy = EnemyBase.new()
	enemy.type = type  # Store type for score tracking
	match type:
		"brown":
			enemy.body_color = Color(0.6, 0.4, 0.2)
		"black":
			enemy.body_color = Color(0.2, 0.2, 0.2)
			enemy.speed = 150.0
		"boss":
			enemy.body_color = Color(1.0, 0.84, 0.0)
			enemy.body_size = Vector2(50, 65)
			enemy.max_health = 10
			enemy.speed = 100.0
	return enemy
```

**After:**
```gd
func _create_enemy(type: String) -> EnemyBase:
	match type:
		"brown": return EnemyBrown.new()
		"black": return EnemyBlack.new()
		"war": return EnemyWar.new()
		"boss": return EnemyBoss.new()
		_: return EnemyBrown.new()
```

Changes:
- Return type changed from `Node2D` to `EnemyBase`
- Removed manual property assignments (each variant sets its own properties)
- Added `EnemyWar` case ("war" type)
- Added fallback to `EnemyBrown.new()` for unknown types
